如何優化程式執行效率

我主要分成兩大部分:

硬體

1. SIMD(Single Instruction, Multiple Data)

2. 多核心/Multithead/OpenMP

3. GPU/CUDA/OPENCL

軟體

1. 數學

2. 演算法技巧

可以利用演算法Pattern來加速程式執行效率
例如: Binary Search, Sliding window,DP, prefix sum..etc
看見這些演算法技巧是否很眼熟呢?沒錯在LeetCode題目中這些都是常見的解題技巧。接下來我會陸續更新各種演算法技巧如何應用於影像處理領域上。

Binary Search: sqrt()
Two Point: Run-Length Encoding Connected Component LabelingCCL
Prefix sum: adaboostfacedetection, NonLocalMean
Sliding windows: StackBlur FastMedianFilter
DP: Stereo image with semi global matching StereoImageSGM
DFS:Connected Component LabelingCCL
Union Find: Connected Component Labeling CCL

3. 資料導向設計

主要概念是將要處理資料盡可能放在連續的記憶體空間來提升記憶體cache hit rate達到提升效率，另一方面如果記憶體連續後比較好配合使用SIMD優化

4. Strength Reduction

a. Lookup Table
sin(x), cos(x) 需先建立起table可加速運算但會失去精準度

b. Bit Shift replace of multiplication/division
用bit shift可以有效率高於乘除法

如果乘數是2^N可調用shift取代
如果除數是2^N可調用shift取代
   var = var / 2;
   // better
   var >>=1;
swap function
   void swap(int *a,int *b)
   {
     int tmp;
     tmp = *a;
     *a =*b;
     *b=tmp
   }
   // better
   void swap(int *a,int *b)
   {
     *a^=*b;
     *b^=*a;
     *a^=*b;
   }
奇偶數判斷
   if(var % 2)
   //better
   if(var & 2)
